# AdGuardHome Block and Allow lists for use with HaGeZi Pro++

**Optimized for:**
- **Echo Show** (100% ad-free)
- **Roku / Apple TV** (30+ streaming apps)
- **NBC Sports, Peacock, Hulu, Disney+, YouTube, Prime Video, HBO Max, Philo, Tubi, etc.**

## Blocklist Used
HaGeZi's Pro++ Blocklist

   ``https://raw.githubusercontent.com/hagezi/dns-blocklists/main/adblock/pro.plus.txt``

## How to Use
1. Add HaGeZi's Pro++ Blocklist to AdGuardHome
2. Save → Enable
3. Add this raw URL to **AdGuard Home** → **Filters** → **DNS blocklists**

   ``https://raw.githubusercontent.com/phantasm22/AdGuardHome-Lists/main/blocklist.txt``

4. Save → Enable
5. Add this raw URL to **AdGuard Home** → **Filters** → **DNS allowlists**
   
   ``https://raw.githubusercontent.com/phantasm22/AdGuardHome-Lists/main/allowlist.txt``

6. Save → Enable
   

## Auto-Updates
These files are version-controlled. Edit → commit → AdGuard updates automatically.
